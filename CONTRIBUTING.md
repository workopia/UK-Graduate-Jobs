# Contributing

Thanks for helping keep this list accurate! 🙏

## How this list is built
This repository is **generated automatically by [Workopia](https://workopia.io)** and refreshed **daily**. Every role is sourced **straight from the employer's official career page / ATS feed** — never scraped from other job boards — and each **Apply →** link opens the role on Workopia, where you get an AI match score, salary insight, a tailored CV & cover letter, and a direct link to the employer's official posting.

The list here is a hand-picked slice. The full, always-fresh list lives at **[workopia.io/browsejobs/uk](https://workopia.io/browsejobs/uk)** (search & filter by role, city, salary and date).

## How to contribute
Because the list is generated from live data, the most useful contribution is **reporting problems**:

- **A role is closed, wrong, mislabelled, or duplicated?** → [Open an issue](../../issues/new/choose) using the "Report a role" form.
- **Suggest a category or country** you'd like added (e.g. Singapore, Australia, US; Legal, Engineering) → open an issue.

We review reports and fix the source data so the daily rebuild picks up the correction.

💻 Tech · 💷 Finance & Accounting · 🏥 Healthcare · 🎓 Graduate Schemes — across **Internship / Graduate / Entry-level** tiers.
